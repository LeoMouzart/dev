
🏆 Desafio: O Analista de Temperaturas
Você foi contratado para criar um pequeno sistema para uma estação meteorológica. O objetivo é ler as temperaturas registradas durante uma semana (7 dias) e gerar um relatório básico.

O que o programa deve fazer:
Entrada: Criar um array de double com tamanho 7.

Coleta: Usar um laço (for) para pedir ao usuário que digite a temperatura de cada um dos 7 dias.

Processamento:

Calcular a Média das temperaturas da semana.

Encontrar a Maior temperatura registrada.

Encontrar a Menor temperatura registrada.
