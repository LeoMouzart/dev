## Variaveis


Variavéis são espaços na memória reservados para receber uma informação.

<br>

Uma variável como o nome já diz pode ter seu valor alterado a qualquer momento.

### Criação de variaveis

<p>

Podemos usar um TIPO de dados ou a palavra reservada <b>var</b> para criar uma variável.<br>
No C# o tipo vem antes da variável!
